# 🩺 Mental Health Risk Prediction System

An AI-based Machine Learning web application that predicts a person's mental health risk level using lifestyle and behavioral patterns.

Built using:
- Python
- Machine Learning
- Decision Tree Classifier
- Streamlit

---

# 🚀 Project Overview

Mental health problems are increasing due to:
- Stress
- Lack of sleep
- Excessive screen time
- Reduced physical activity
- Social isolation

This project predicts mental health risk levels based on daily habits and lifestyle data.

The system classifies users into:
- 🟢 Low Risk
- 🟡 Medium Risk
- 🔴 High Risk

The goal is to help users become aware of unhealthy habits and encourage better lifestyle management.

---

# 📊 Features Used

The model analyzes the following inputs:

| Feature | Description |
|---|---|
| Sleep Hours | Daily sleep duration |
| Stress Level | Low / Medium / High |
| Screen Time | Daily device usage |
| Exercise Hours | Physical activity duration |
| Social Interaction | Social engagement level |
| Water Intake | Daily water consumption |
| Work Pressure | Stress from work/study |
