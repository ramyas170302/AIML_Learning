# 🏦 Loan Approval Prediction System

## 📌 Project Overview
This project is a Machine Learning web application that predicts whether a loan application will be **Approved or Rejected** based on applicant financial and personal details.

The model is built using **Random Forest Classifier** and deployed using **Streamlit**.

---

## 📊 Problem Statement
Banks receive many loan applications daily. Manual verification is time-consuming and may lead to errors.  
This project automates loan approval prediction using Machine Learning.

---

## 🧠 Machine Learning Model
- Algorithm: Random Forest Classifier
- Library: Scikit-learn
- Task: Binary Classification (Approve / Reject)

---

## 📂 Dataset
- Source: Kaggle Loan Approval Dataset  
- Features include:
  - Income
  - Loan Amount
  - CIBIL Score
  - Residential Assets
  - Commercial Assets
  - Luxury Assets
  - Bank Assets
  - Education
  - Employment Status
