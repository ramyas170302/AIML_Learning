# 🎓 AI Placement Mentor

Your Personal AI Coach for Placement Preparation 🚀

## 📖 About the Project

AI Placement Mentor is an AI-powered web application that helps engineering students prepare for placements more effectively.

The application generates personalized career roadmaps, interview questions, answer reviews, and study plans using Large Language Models (LLMs) through LangChain and Groq.

---

## ✨ Features

### 🛣️ Career Roadmap

* Personalized roadmap based on career goal
* Skill gap analysis
* Recommended technical subjects
* Project suggestions
* Weekly learning plan
* Placement preparation tips

### 💻 Interview Question Generator

* Technical, HR, and Behavioral questions
* Multiple difficulty levels
* Coding, Theory, MCQ, and Mixed questions
* Company-specific interview preparation

### ✅ Answer Review

* Reviews user answers
* Gives score out of 10
* Highlights strengths
* Identifies mistakes
* Suggests improvements
* Provides an ideal answer

### 📅 Study Planner

* Personalized daily study plan
* Weekly schedule
* Learning priorities
* Practice recommendations
* Revision schedule
* Mini project suggestions
